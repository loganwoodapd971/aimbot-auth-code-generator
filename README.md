## Overview

Aimbot License Generator is a small Android-focused utility for creating authorization codes through a browser UI. It ships as a lightweight HTML package, so it can be hosted locally without relying on any backend component.

The project is built for self-managed deployments where a straightforward offline process and direct control over generated codes are important. With support for 48-hour validity, it offers a narrow, practical workflow for short-lived authorization in a minimal footprint.
